3D artwork done in Blender

The 64-bit version is compiled using MSYS and CMake for modern systems

The 32-bit version is compiled using VS2017 and v141_xp for Windows XP and older systems

You can find more information and older versions at the Github page

Features

  • Snappy image controls + zoom
  • Draw mode
  • Color, size, opacity (alpha blending), eyedropper, and softness
  • Automatic Adjustment
  • Brightness/Contrast
  • Invert Colors
  • Gaussian Blur
  • Draw Text
  • Crop Image
  • Delete, print, copy, and paste
  • Zoom auto / Zoom 1:1 scale
  • Lightweight and performant
  • File support
  • Uses stb_image formats: JPEG, PNG, TGA, BMP, PSD (composited only), static GIF, HDR (only sRGB), PIC, and PNM
  • Specialty supports include SVG, SFBB, M45, and IRBO
